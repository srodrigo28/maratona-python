## Roadmap - Curso Python do Básico ao Intermediário

**Público-alvo**: Iniciantes sem experiência ou com pouquíssimo contato com programação  
**Duração estimada**: 12–16 semanas (2–3 aulas por semana)  
**Total de módulos**: 12  
**Formato sugerido**: Videoaulas + exercícios práticos + mini-projetos em cada módulo

### Módulo 0 - Preparação do Ambiente (1 aula)
- Por que aprender Python em 2025/2026
- Instalação do Python (Windows, macOS, Linux)
- VS Code + extensões essenciais
- Google Colab (alternativa sem instalação)
- Primeiro "Hello, World!"
- Como rodar scripts (.py) e usar o REPL

### Módulo 1 - Fundamentos da Linguagem (4–5 aulas)
- Variáveis e tipos de dados (int, float, str, bool)
- Entrada e saída de dados (input/print)
- Operadores aritméticos, relacionais e lógicos
- Comentários e boas práticas de nomeação
- Exercícios: calculadora simples, conversor de unidades

### Módulo 2 - Estruturas de Controle de Fluxo (4 aulas)
- if, elif, else
- match-case (Python 3.10+) - introdução leve
- Loops: while e for
- range(), break, continue, else em loops
- Exercícios: adivinhação de número, tabuada, contagem regressiva

### Módulo 3 - Estruturas de Dados Básicas (5–6 aulas)
- Listas (list) - métodos importantes
- Tuplas (tuple)
- Dicionários (dict) - acesso, atualização, métodos
- Sets (conjuntos) - operações matemáticas
- Fatiamento (slicing) em sequências
- List/dict/set comprehensions (introdução)
- Mini-projeto: Gerenciador de tarefas (CLI) ou lista de compras

### Módulo 4 - Funções e Modularização (4 aulas)
- Definindo e chamando funções
- Parâmetros e retorno (return)
- Escopo de variáveis
- Funções anônimas (lambda)
- Args e kwargs
- Módulos e importação (criar seus próprios módulos)
- Exercícios: calculadora com funções, validação de CPF simples

### Módulo 5 - Manipulação de Strings e Textos (3 aulas)
- Métodos de string (.upper, .split, .join, .strip, etc.)
- Formatação de strings (f-strings, .format)
- Expressões regulares básicas (módulo re) - opcional
- Exercícios: validador de e-mail, contador de palavras, criptografia César

### Módulo 6 - Manipulação de Arquivos (3 aulas)
- Leitura e escrita de arquivos txt
- Modos de abertura (r, w, a, r+)
- Trabalhando com CSV (módulo csv)
- Trabalhando com JSON (módulo json)
- Mini-projeto: Sistema de cadastro que salva em JSON/CSV

### Módulo 7 - Tratamento de Exceções (2 aulas)
- try, except, else, finally
- Hierarquia de exceções
- Levantando exceções (raise)
- Criando exceções personalizadas (básico)
- Exercício: melhorar projetos anteriores com tratamento de erro

### Módulo 8 - Programação Orientada a Objetos - Parte 1 (5 aulas)
- Conceitos: classe, objeto, atributos, métodos
- Método __init__ e self
- Encapsulamento básico
- Métodos de instância, de classe e estáticos
- Mini-projeto: Sistema bancário simples (Conta, Cliente)

### Módulo 9 - Programação Orientada a Objetos - Parte 2 (4 aulas)
- Herança e polimorfismo
- super()
- Propriedades (@property)
- Métodos mágicos (__str__, __repr__, __len__, etc.)
- Composição vs herança
- Mini-projeto: Jogo de texto (Personagem, Inimigo, Item)

### Módulo 10 - Bibliotecas Essenciais do Ecossistema (5–6 aulas)
- datetime - datas e horários
- os e pathlib - sistema de arquivos
- requests - consumo de APIs REST
- random e secrets
- collections (Counter, defaultdict, namedtuple)
- itertools (básico)
- Projeto prático: Consumir uma API pública (ex: Pokémon, Chuck Norris, clima)

### Módulo 11 - Testes Automatizados (2–3 aulas)
- Importância dos testes
- unittest básico
- pytest (instalação e primeiros testes)
- Cobertura de código (opcional)
- Exercício: escrever testes para o sistema bancário

### Módulo 12 - Projeto Final + Próximos Passos (3–4 aulas)
- Escolha entre 3 projetos finais sugeridos:
  1. Bot de automação (web scraping + selenium ou API do WhatsApp/Telegram)
  2. Aplicação web simples com Flask
  3. Análise de dados com pandas (básico)
- Apresentação e revisão do código
- Boas práticas finais (PEP 8, docstrings, type hints introdutório)
- Roadmap de estudos avançados (FastAPI, Django, Data Science, Automação, etc.)

## Bônus (entregue ao longo do curso ou no final)
- Dicas de Git e GitHub para iniciantes
- Como criar um portfólio
- Virtual environments (venv) e gerenciadores (poetry/pipenv)
- Type hints básicos
- Debug no VS Code

## Estrutura sugerida de cada aula
1. Teoria (15–25 min)
2. Código ao vivo (20–30 min)
3. Exercícios guiados
4. Desafio para casa (com gabarito na próxima aula)
5. Mini-projeto a cada 2–3 módulos

Com esse roadmap você tem um curso extremamente completo, atualizado e com alta empregabilidade no mercado brasileiro. Boa sorte na criação do curso!
